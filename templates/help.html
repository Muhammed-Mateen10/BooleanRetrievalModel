Boolean Queries

A query that matches documents matching boolean combinations of other queries. The clause (query) must appear in matching documents and will contribute to the score. Boolean search is a type of search allowing users to combine keywords with operators (or modifiers) such as AND, NOT and OR to further produce more relevant results. For example, a Boolean search could be “hotel” AND “New York”. This would limit the search results to only those documents containing the two keywords.

Proximity Query

In text processing , a proximity search looks for documents where two or more separately matching term occurrences are within a specified distance Example: query = "text model /2" Documents Containing text and model atmost 2 distance apart would be returned
